# webview/components/dialogs/ - Webview Dialog Components

Modal dialog components for content insertion within webview editors.

## Files

- **InsertLinkDialog.tsx** - Dialog for inserting or editing hyperlinks with URL and link text input fields. Used by the Lexical editor.
- **InsertYouTubeDialog.tsx** - Dialog for inserting YouTube videos that accepts various URL formats (full URL, short URL, embed URL) and extracts the video ID.
- **index.ts** - Export barrel for dialog components.
