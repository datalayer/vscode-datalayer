# webview/styles/ - CSS Style Overrides

CSS files for VS Code theme integration and component styling overrides.

## Files

- **vscode-primer-overrides.css** - Themes Primer React ActionMenu components with VS Code colors using CSS variables (`--vscode-menu-background`, `--vscode-menu-foreground`, etc.). Ensures dropdown menus match the active VS Code theme.
- **vscode-completion-theme.css** - Styles for inline completion ghost text and completion dropdown UI. Integrates with VS Code's theme for consistent appearance.
- **collapsible-vscode.css** - Styles for collapsible sections in the webview matching VS Code's native collapsible panel appearance.
