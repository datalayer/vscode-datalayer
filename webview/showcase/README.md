# webview/showcase/ - Primer Theme Showcase

Development tool for previewing Primer React components with VS Code theme integration.

## Files

- **PrimerShowcase.tsx** - Renders a gallery of Primer React components (buttons, inputs, labels, etc.) styled with VS Code theme colors. Used during development to verify theme mapping correctness.
- **index.tsx** - Entry point exporting the PrimerShowcase component.
