# src/test/models/ - Model Unit Tests

Unit tests for tree view data models used in the extension sidebar.

## Files

- **datasourceTreeItem.test.ts** - Tests for DatasourceTreeItem: label, description, tooltip content, icon, context value, and click command.
- **runtimeTreeItem.test.ts** - Tests for RuntimeTreeItem: label fallback, time remaining formatting, environment display, tooltip, and icon.
- **secretTreeItem.test.ts** - Tests for SecretTreeItem: label, masked value display, tooltip security, and icon.
- **snapshotTreeItem.test.ts** - Tests for SnapshotTreeItem: label, environment description, relative time display, tooltip, and icon.
- **spaceItem.test.ts** - Tests for SpaceItem: item type enum values, tooltips per type, icons per type, document icon mapping by file extension, and command assignment.
- **treeSectionItem.test.ts** - Tests for TreeSectionItem: section header label, collapsible state, and icon.
