# src/ui/ - UI Components

UI components for dialogs, selectors, styles, and HTML templates used by the extension.

## Subdirectories

- **dialogs/** - Modal dialogs and input prompts (auth, confirmation, credentials, kernel/runtime selection)
- **selectors/** - QuickPick-based selection UIs
- **styles/** - CSS theme mappings (VS Code to Primer)
- **templates/** - HTML templates for webview panels
