# src/ui/dialogs/ - Modal Dialogs

Dialog components for user interaction including authentication, confirmation, and selection flows.

## Files

- **authDialog.ts** - Non-intrusive authentication notification dialog with convenience function combining dialog with login action.
- **authMethodSelector.ts** - Authentication method selection supporting email-password and OAuth methods (GitHub, LinkedIn).
- **confirmationDialog.ts** - Two-step confirmation for destructive actions using non-intrusive notifications with clear consequence listing.
- **credentialsInput.ts** - Two-step credential input dialogs for handle and password with validation functions.
- **kernelSelector.ts** - Kernel selection menu with options for different kernel types including local and cloud kernels.
- **runtimeSelector.ts** - Runtime selection UI for choosing existing runtimes or creating new ones with configuration options.
